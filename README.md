# File I/O demo

This repository contains a small Python file I/O utility and a demo.

Files:

- `file_io.py`: helper functions for reading/writing/appending text files.
- `main.py`: simple CLI demo to add and list names stored in `names.txt`.
- `names.txt`: example data file.

Usage:

Run the demo:

```bash
python main.py
```

The demo can add names to `names.txt` and list existing names.
